# Contributing

Thanks for checking out PingVista.

## Local Setup

```bash
npm start
```

Then open:

```text
http://127.0.0.1:4175
```

## Checks Before Opening A Pull Request

```bash
npm run check
npm run test:security
```

## Contribution Ideas

- Improve deployment guides
- Add tests for edge cases
- Improve accessibility
- Add alert integrations
- Add Docker support
- Improve Supabase setup docs

## Style Notes

- Keep the project dependency-light.
- Keep public deployment safety limits visible.
- Avoid removing SSRF protections or rate limits.
- Prefer clear UI copy over marketing-heavy language.
